About GrainPalette – Rice Type Classifier

GrainPalette is a deep learning–powered rice classification system. It uses image recognition through Convolutional Neural Networks (CNNs) and MobileNet (transfer learning) to identify the rice grain type from a photo.

The purpose of this project is to assist farmers, agri-scientists, and researchers by:

  • Identifying rice types quickly and accurately
  • Helping with proper irrigation and fertilizer decisions
  • Enabling data-driven agricultural practices

The model was trained on high-quality rice grain images, built and optimized using Google Colab with TensorFlow and Keras. It is deployed as a Flask web app, allowing users to upload grain images and receive instant predictions.

This innovation empowers precision farming and enhances grain quality monitoring in a scalable, automated way.
